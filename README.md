# Study Planner

A web-based Study Planner application built using Python and Flask to help students organize their study schedules, manage tasks, and improve productivity.

## Features

* Add and manage study tasks
* Organize daily study schedules
* Track task completion
* Simple and user-friendly interface
* Local database storage using SQLite

## Technologies Used

* Python
* Flask
* HTML
* CSS
* SQLite
* Git & GitHub

## Installation

1. Clone the repository

```bash
git clone https://github.com/deetshithabhavani/study-planner-.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python app.py
```

4. Open in browser

```text
http://127.0.0.1:5000
```
